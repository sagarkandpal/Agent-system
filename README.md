# AI Agents 🤖

This repository contains AI Agent projects built using Python, LangChain, Tools, LLMs, and Multi-Agent workflows.

AI Agents can think, reason, use tools, search the web, scrape data, and perform tasks autonomously using Large Language Models.

---

# 🚀 Features

- Tool Calling
- Web Search Agents
- Multi-Agent Systems
- Research Agents
- Memory Support
- AI Workflows
- Autonomous Task Execution
- LangChain Integration

---

# 🛠 Tech Stack

- Python
- LangChain
- OpenAI / Groq / Mistral
- Tavily Search
- BeautifulSoup
- Streamlit
